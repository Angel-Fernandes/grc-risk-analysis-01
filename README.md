# Análise de Riscos em GRC

## 📌 Descrição
Projeto de análise de riscos em Segurança da Informação.

## 🧩 Ativos
- Aplicativo web e mobile
- Banco de dados
- E-mail
- Firewall
- Monitoramento

## ⚠️ Ameaças
- Ataques cibernéticos
- Erros humanos
- Ausência de backups

## 🚨 Riscos
- Controle de acesso inadequado
- Falta de políticas de segurança
- Falta de continuidade

## 📊 Matriz de Risco
Todos os riscos foram classificados como ALTO.

## 🧠 Conclusão
Ambiente com baixa maturidade em segurança.
